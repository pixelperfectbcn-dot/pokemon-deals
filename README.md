# Pokemon Deals Radar

Proyecto completo base para Railway con:
- Frontend Next.js
- API interna Next.js
- Endpoints:
  - GET /api/health
  - GET /api/deals
  - POST /api/refresh

## Uso local

```bash
npm install
npm run dev
```

## Deploy en Railway
1. Sube TODO este contenido a la raíz del repo.
2. Asegúrate de que en la raíz del repo se ven:
   - app/
   - lib/
   - package.json
3. Railway:
   - Build Command: npm run build
   - Start Command: npm run start
